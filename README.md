#  linux-essentials

##  Sobre
Repositório dedicado ao registro de estudos, anotações e laboratórios práticos sobre os fundamentos do sistema operacional Linux, a base essencial para soluções em Cloud e cultura DevOps.

##  Objetivo
Documentar a evolução nos conceitos de administração de sistemas, comandos de terminal, gerenciamento de permissões e automação de tarefas.

##  Conteúdo Programado
- `comandos-basicos`: Navegação, manipulação de arquivos e busca.
- `usuarios-e-permissoes`: Gestão de privilégios (chmod, chown, sudo).
- `processos`: Monitoramento e controle de execução do sistema.
- `scripts`: Automações simples e Shell Script.
- `anotacoes`: Resumos teóricos e dicas de terminal.

##  Ferramentas Utilizadas
- Linux (Ubuntu / Debian / CentOS)
- Git & GitHub
- Bash Terminal
